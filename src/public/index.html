Sonic

The Hedgehog Games API

Acá podrás encontrar toda la información relacionada a los juegos de nuestro erizo azul favorito.
Además de otras utilidades que te ayudarán a completar la información que necesites para el desarrollo de tu App.
Esta API es de uso libre y no requiere clave de acceso para el uso de la misma.

Endpoints

Juegos

Este endpoint brinda un un objeto JSON con toda la información relacionada a los juegos de Sonic The Hedgehog, desde el más viejo hasta el más nuevo.

https://nelsongamerodev.com/sonic-games-api/games/

Cada juego tiene su propio ID único representado por un número y puede ser usado para una consulta específica de ese juego. Simplemente debemos indicar dicho ID cómo una ruta.

https://nelsongamerodev.com/sonic-games-api/games/2

Se puede realizar una consulta más específica pasando por query params los valores "platform" y "genre". De esta forma la lista de juego final será filtrada en base a esos parámetros. No es necesario indicar los 2 valores, puede ser simplemente uno u otro. En caso de que algún valor tenga un espacio entre medio (ej: playstation 3) se debe reemplazar por un guión (ej: playstation-3). La búsqueda de valores no debe ser exacta, por lo que no es necesario completar las palabras de la plataforma o el género que se requiera, basta con poner 3 letras para que el arreglo de juegos se filtre en base a las concidencias encontradas.

https://nelsongamerodev.com/sonic-games-api/games?platform=playstation-3&genre=aventura

https://nelsongamerodev.com/sonic-games-api/games?platform=play&genre=aven

Plataformas

Este endpoint brinda un un objeto JSON con todas las plataformas de los juegos

https://nelsongamerodev.com/sonic-games-api/platforms/

Géneros

Este endpoint brinda un un objeto JSON con todos los géneros de los juegos

https://nelsongamerodev.com/sonic-games-api/genres/
